|               快捷键 | 作用                         |
| -------------------: | :--------------------------- |
|  shift + ctrl + 空格 | 代码提示                     |
|       ctrl + alt + V | 补全代码 / 提出代码为变量    |
| shift + ctrl + enter | 补全方法体                   |
|        shift + enter | 向下插入一行，光标也随之移动 |
|   ctrl + alt + enter | 向上插入一行，光标也随之移动 |
|         ctrl + enter | 向下插入一行，光标位置不动   |
|           ctrl + F12 | 查看类中所有方法             |
|             ctrl + O | 快速重写方法                 |
|             ctrl + I | 快速实现方法                 |
|              alt + 1 | 收起或展示Project栏          |
|       ctrl + alt + O | 清理无用包                   |
|       ctrl + alt + L | 整理代码格式                 |
|             ctrl + B | 进入实现类，子类，底层源码   |
|       ctrl + alt + T | 快速生成try-catch等代码块    |
|       ctrl + alt + ← | 跳出方法到调用者             |
|       ctrl + alt + → | 进入方法                     |
|       ctrl + alt + u | 查看类图                     |
